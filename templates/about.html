About Dorset Music Collection



Dorset Music Collection (DMC) is a web app developed for my Continuous Assessment 1 at Dorset College BSc in Computing, Year 3, Semester 1.


The backend code for this project was written in Python, using Django. At the moment, the website only offers one app: Polls, which allows users to create and vote in polls to elect the best album of a specific music genre out of the TOP-10 most popular released in a specific year. The data is pulled from Discogs* through their API.

Visit the project's GitHub.



*This application uses Discogs’ API but is not affiliated with, sponsored or endorsed by Discogs. ‘Discogs’ is a trademark of Zink Media, LLC.
